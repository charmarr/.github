# Support

Welcome to Charmarr! Here's how to get help.

## Documentation

Start with the documentation:

- **Project README**: Each repository has a README with usage instructions
- **Contributing Guide**: See [CONTRIBUTING.md](CONTRIBUTING.md) for development help
- **charmarr-meta**: Architecture Decision Records (ADRs) and project documentation

## Community Support

### GitHub Discussions

For questions, ideas, and general discussion:

**[GitHub Discussions](https://github.com/orgs/charmarr/discussions)**

This is the best place to:
- Ask "how do I...?" questions
- Share your Charmarr setup
- Discuss feature ideas
- Get help troubleshooting

### GitHub Issues

For bugs and feature requests:

**[Repository Issues](https://github.com/charmarr/[repo]/issues)**

Use issues to:
- Report bugs
- Request new features
- Report documentation problems

Please check existing issues before creating a new one.

## Juju Charm Support

For Charmarr Juju charms:

1. **Check charm README**: Each charm has detailed deployment and configuration docs
2. **Juju logs**: Run `juju debug-log` to see charm logs
3. **Charm status**: Run `juju status --relations` to check the state
4. **Integration tests**: Charms include BDD tests demonstrating common scenarios

## Before Asking for Help

To get the fastest response:

1. **Check the README** for the specific repository
2. **Search existing issues and discussions** - your question may already be answered
3. **Include details** when asking:
   - What you're trying to do
   - What you expected to happen
   - What actually happened
   - Relevant logs, error messages, or screenshots
   - Your environment (OS, Kubernetes version, Juju version, etc.)

## Response Time

Charmarr is an open-source project maintained by volunteers. Response times vary:

- **Community support**: Best effort, typically within days
- **Security issues**: See [SECURITY.md](SECURITY.md) for faster response

## Contributing

The best way to ensure your needs are met is to contribute! See [CONTRIBUTING.md](CONTRIBUTING.md) for how to get started.
